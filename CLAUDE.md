# The Heartbeat - Book & Sales Consulting Website

## Project Overview
Website for "The Heartbeat: Anatomy of a Sale" book by a sales consultant/coach. Serves as:
- Book promotion and sales page
- Sales consulting & coaching services
- Speaking engagement bookings
- Corporate and individual training programs

## Tech Stack
- Next.js 16 (App Router, TypeScript)
- Tailwind CSS v4
- Deployed to Vercel

## Design System
- **Primary color**: Deep red/crimson (#B91C1C / red-700) — heartbeat theme
- **Accent**: Warm neutrals, charcoal text
- **Font**: Geist Sans (loaded via next/font)
- **Tone**: Professional, warm, authoritative

## Site Structure
- `/` — Homepage (hero, book intro, services overview, testimonials, CTA)
- `/book` — Book details, chapter overview, purchase links
- `/services` — Consulting, coaching, speaking, training
- `/contact` — Contact form for hiring/booking

## Commands
- `npm run dev` — Start dev server
- `npm run build` — Production build
- `npm run lint` — Run ESLint
